# HTML & CSS Self Learning Tasks

A collection of HTML & CSS exercises completed as part of a self learning training.

## Setup

1. Clone the repo:
```bash
   git clone https://github.com/ananyahegde/HTML_and_CSS.git
```
2. Navigate into a task folder (e.g., task-1):
```bash
   cd task-1
```
3. Open `index.html` in your browser:
```bash
   google-chrome index.html
```

## Tasks

1. Static Webpage Layout
   - A single-page website using HTML semantic elements
   - CSS for basic styling and spacing
   - Responsive design for smaller screens using media queries

2. CSS Card Component with Hover Effects
   - Simple card component with image, title and description
   - CSS borders and shadows for card styling
   - CSS hover effect for background change

3. Responsive Grid Layout
   - Simple CSS grid layout
   - Changes from horizontal to vertical stack on smaller screens

4. Pure CSS Dropdown Menu
   - A navigation bar with dropdown submenu
   - CSS hover effects on navbar items
   - Media query to change navbar appearance on smaller screens

5. Modal Popup Using `:target` or Checkbox Hack
   - A simple popup on button click in pure CSS
   - CSS to control modal visibility based on checkbox's checked state
   - CSS backdrop filter on modal popup
   - CSS transitions for smooth modal display

6. Tabbed Content Interface
   - Show different sections based on tab click
   - CSS transitions for smooth effect when switching between tabs

7. Pure CSS Carousel/Slider
   - Image carousel with manual slide selection and automatic sliding
   - CSS `:checked` hack to control individual image visibility
   - CSS animations (`@keyframes`) for automatic slideshow
   - CSS transitions for visual appeal

8. CSS Accordion Component
   - Pure CSS accordion component
   - CSS checkbox hack to control the open/closed state of each item

9. Complex Responsive Layout with Grid and Flexbox
   - Combines CSS Grid and Flexbox to create a complex website structure
   - Responsive layout changes based on viewport size
   - Sticky sidebar for navigation

10. Interactive Multi-Page Website Simulator with CSS Only
    - Multi-section, single-page website with pure CSS
    - Sticky navbar with responsive design
